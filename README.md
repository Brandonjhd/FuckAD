## 🇨🇳 简体中文

自动合并多源广告拦截规则 —— 适合规则维护者与重度洁癖用户的轻量工具

### 简介

FuckAD 是一个用于自动获取并合并广告拦截规则的开源工具。  
它会从多个远程规则源拉取最新内容，自动过滤注释和空行，对规则进行去重和排序，最终生成统一的规则文件 `FuckAD.conf`，方便作为订阅源或进一步分发使用。

当前默认规则源包括：

- https://adrules.top/adrules-surge.conf
- https://whatshub.top/rule/AntiAD.list

### 主要功能

- 自动从多个远程规则源获取最新广告拦截规则
- 忽略注释与空行，仅保留有效规则行
- 合并、去重并按字典序排序所有规则
- 在输出文件开头添加更新时间、总规则数及致谢信息
- 可通过 GitHub Actions 定期自动更新并提交

### 使用方式

- 克隆本仓库到本地
- 确保已安装 Python 3.11 或更高版本
- 在仓库根目录执行 `python fuck_ad_rule.py` 生成或更新 `FuckAD.conf`
- 也可以直接使用仓库中由 GitHub Actions 自动生成的 `FuckAD.conf` 作为远程规则订阅

## 🇺🇸 English

Automatically merge ad-blocking rules from multiple sources — a lightweight tool for rule maintainers and power users.

### Introduction

FuckAD is an open-source tool that automatically fetches and merges ad-blocking rules from multiple remote sources.  
It filters out comments and empty lines, deduplicates and sorts all rules, and finally generates a unified rules file `FuckAD.conf` for use as a subscription source or further distribution.

Current default rule sources:

- https://adrules.top/adrules-surge.conf
- https://whatshub.top/rule/AntiAD.list

### Features

- Automatically fetch the latest ad-blocking rules from multiple remote sources
- Ignore comments and empty lines, keeping only valid rule entries
- Merge, deduplicate, and sort all rules in lexicographical order
- Add a header with update time, total rule count, and acknowledgements
- Can be scheduled with GitHub Actions for periodic automatic updates and commits

### Usage

- Clone this repository to your local machine
- Ensure Python 3.11 or later is installed
- Run `python fuck_ad_rule.py` in the repository root to generate or update `FuckAD.conf`
- Alternatively, directly use the `FuckAD.conf` file automatically generated by GitHub Actions in this repository as a remote rules source

### 概要

FuckAD は、複数のリモートソースから広告ブロックルールを自動取得して統合するオープンソースツールです。  
コメント行や空行を除外し、重複排除とソートを行った上で、統一されたルールファイル `FuckAD.conf` として出力します。サブスクリプションや配布に便利です。

現在のデフォルトルールソースは次のとおりです。

- https://adrules.top/adrules-surge.conf

- 複数のリモートソースから最新の広告ブロックルールを自動取得

- 出力ファイルの先頭に更新時刻、ルール総数、謝辞情報を自動付加
